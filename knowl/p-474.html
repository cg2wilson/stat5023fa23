Based on the SAS output (salt_output(1).pdf), we see the following (for VC):
  • The "Estimated R Matrix" identifies the relationship between the four time roles; this indicates that \(\sigma^2 = 56.5212\text{;}\) this is confirmed in the "Covariance Parameter Estimates" table.
  • The AIC and BIC are going to be compared between different structures to determine the best.
  • The Null Model Likelihood Ratio Test tests the hypothesis
    \begin{equation*} H_0: \text{VC is appropriate } \end{equation*}
    versus
    \begin{equation*} H_a: \text{the other is appropriate} \end{equation*}
    So since this is VC vs. VC, this table contributes nothing.
  • The Fixed Effects table indicates that there is significant interaction, so we proceed normally.
For AR(1), we have
  • In the "Covariance Parameter Estimates" table, the AR(1) line indicates \(\hat{\rho}\text{,}\) and the Residual indicates \(\hat{\sigma}^2\text{.}\) In particular, we see that \(\hat{\rho}\cdot \hat{\sigma}^2 = 0.83\cdot 52.7691 = 43.7987\text{.}\) Similarly, \(\hat{\rho}^2\sigma^2 = (0.83)^2\cdot 55.7691=36.3533\)
  • Comparing the AIC and BIC values with those from VC (215.6 vs. 245.4 and 216.6 vs. 245.9), since we go with smaller values, this suggests that AR(1) is more appropriate.
  • The NMLR test concludes that \(p \lt 0.0001\text{,}\) so between the AIC/BIC measurements and the NMLR test, we choose AR(1) as the appropriate structure.
For CS, we have
  • Since CS assumes covariance is the same, we see this in the R Matrix and the Parameter Estimates tables. Similarly, we see that \(Cov(X,Y) + \sigma^2 = Cov(X,X) \text{.}\)
  • We see that AR(1) has smaller AIC and BIC values than CS, so we prefer AR(1) to CS, and both over VC.
  • NMLR concludes that CS is more appropriate than VC.
