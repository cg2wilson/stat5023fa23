Step-by-step analysis goes as follows:
  1. Is the covariate necessary? Test
    \begin{equation*} H_0: \beta_1 = \cdots = \beta_t = 0 \end{equation*}
    vs.
    \begin{equation*} H_a: \beta_j \neq 0 \text{ for some } j \end{equation*}
    If we fail to reject \(H_0\text{,}\) we proceed with one-way ANOVA. If we reject \(H_0\text{,}\) proceed to step 2.
  2. Are the lines parallel? Test
    \begin{equation*} H_0: \beta_1 = \cdots = \beta_t \end{equation*}
    vs.
    \begin{equation*} H_a: \text{ at least one }\beta_i\text{ is distinct} \end{equation*}
    If we fail to reject \(H_0\text{,}\) we adopt the parallel lines model. Then, each treatment effect can be compared at any value of the covariate. This is equivalent to testing
    \begin{equation*} H_0: \alpha_1 = \cdots = \alpha_t \end{equation*}
    vs.
    \begin{equation*} H_a: \text{ at least one }\alpha_i \text{ is distinct} \end{equation*}
    If we reject \(H_0\text{,}\) we adopt the non-parallel lines model. Then, each treatment effect needs to be compared at different values of the covariates, which is equivalent to testing
    \begin{equation*} H_0: \alpha_1 + \beta_1x_0 = \cdots = \alpha_t + \beta_t x_0 \end{equation*}
    vs.
    \begin{equation*} H_a:\text{ at least one }\alpha_i + \beta_i x_0\text{ is distinct} \end{equation*}
    where \(x_0\) is some specific value of \(X\text{.}\)
