If we have \(t = 4\) treatments and are interested in \(H_0: \mu_2 = \mu_3 = \mu_4\text{,}\) we can develop the mutually orthogonal contrasts
\begin{align*} L_1 \amp = \mu_2 - \mu_3 \\ L_2 \amp = \mu_2 + \mu_3 - 2\mu_4 \end{align*}
The test statistic for \(H_0: L_1 = L_2 = 0\) can be given by
\begin{equation*} F = \dfrac{SS_{H_0}/df_{H_0}}{MSE} = \dfrac{(SSC_{L_1} + SSC_{L_2})2}{MSE}\approx F_{df_{H_0},n_\cdot - t} \end{equation*}
under \(H_0\text{,}\) where MSE comes from one-way ANOVA. We still reject \(H_0\) if \(F \geq F_{\alpha,df_{H_0},n_\cdot - t}\text{.}\)
