We reject \(H_0: \beta_1 = \beta_2 = \beta_3 = 0\) (\(p \lt 0.001\)) at the \(\alpha = 0.05\) significance level. We conclude that height is a significant covariate which as an effect on yield in at least one treatment level.
To test \(H_0: \beta_1 = \beta_2=\beta_3\text{,}\) add MODEL yield = trt height trt*height to the SAS code. This gives the following table:
The \(p-\)value on the last row tests
\begin{equation*} H_0: \beta_1 - \beta_3 = \beta_2 - \beta_3 = 0 \end{equation*}
We fail to reject \(H_0\) under \(\alpha = 0.05\text{,}\) so a parallel lines model is most appropriate.
