Here are some useful formulas to construct the test statistic. When \(Y_1,Y_2,...,Y_n\) is a random sample from a population with variance \(\sigma^2\text{,}\) we have
  • \(\displaystyle \text{Var}\lrpar{\sum_{i=1}^n Y_i} = \sum_{i=1}^n \text{Var}(Y_i) = \sum_{i=1}^n \sigma^2 = n\sigma^2\)
  • \(\displaystyle \text{Var}\lrpar{\sum_{i=1}^n a_iY_i} = \sum_{i=1}^na_i^2\text{Var}(Y_i) = \sigma^2\sum_{i=1}^n a_i^2\)
  • \(\displaystyle \text{Var}(\overline{Y}) = \text{Var}\lrpar{\dfrac{1}{n}\sum_{i=1}^n Y_i} = \sum_{i=1}^n \dfrac{1}{n^2}\text{Var}(Y_i) = \dfrac{1}{n^2}\cdot n\sigma^2 = \dfrac{\sigma^2}{n}\)
These formulas hold without normality as well.
