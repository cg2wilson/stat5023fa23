First, we set up the appropriate contrasts for each trend:
\begin{align*} L_1 \amp = -2\mu_1 - \mu_2 + \mu_4 + 2\mu_5 \\ L_2 \amp = 2\mu_1 - \mu_2 - 2\mu_3 - \mu_4 + 2\mu_5 \end{align*}
by using the orthogonal polynomials table. Second, we compute each \(SSC_{L_i}\text{.}\)
\begin{equation*} SSC_{L_1} = \dfrac{\lrpar{-2(0.619) - 0.772+1.009+2(1.041)}^2}{\dfrac{1}{4}\lrpar{(-2)^2 + (-1)^2 + 1^2 + 2^2}} = 0.4674 \end{equation*}
