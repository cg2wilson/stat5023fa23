In (1), if we fail to reject \(H_0\text{,}\) there is no significant evidence of linear trend; if we reject, then a significant evidence of linear trend. In (2), if we fail to reject \(H_0\text{,}\) then there is no significant evidence of quadratic trend; thus, we conclude that only linear trend would exist. If we reject \(H_0\text{,}\) this indicates that a quadratic trend would exist.
