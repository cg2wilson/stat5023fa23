When the interaction is significant, compare means among each treatment combination for main effects. So, with
\begin{equation*} L = \ds \sum_{i,j} a_{ij}\mu_{ij} = 0 \end{equation*}
consider
\begin{equation*} H_0: L = 0 \end{equation*}
vs.
\begin{equation*} H_a: L\neq 0 \end{equation*}
\(\hat{L} = \ds \sum_{i,j} a_{ij}\overline{Y_{ij\cdot}}\) estimates \(L\) and its standard error estimates \(s_{\hat{L}}\text{:}\)
\begin{equation*} s_{\hat{L}} = \sqrt{MSE\cdot \sum_{i,j} \dfrac{a_{ij}^2}{n}} = \sqrt{\dfrac{MSE}{n}\cdot \sum_{i,j} a_{ij}^2} \end{equation*}
We have two test statistics:
\begin{equation*} t = \dfrac{\hat{L}}{s_{\hat{L}}} \end{equation*}
which indicates that we reject \(H_0\) if \(|t|\geq t_{\alpha/2,ab(n-1)}\text{,}\) and
\begin{equation*} F = t^2 = \lrpar{\dfrac{\hat{L}}{s_{\hat{L}}}}^2 = \dfrac{SSC_{L}/1}{MSE} \end{equation*}
which indicates that we reject \(H_0\) if \(F \geq F_{\alpha,1,ab(n-1)}\)
