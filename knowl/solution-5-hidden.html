Here, note the following for the first row :
  • \(\displaystyle \hat{L}_1 = \overline{Y}_{1\cdot} - \dfrac{1}{4}\lrpar{\sum_{i=2}^5 \overline{Y}_{i\cdot}}\)
  • \(\displaystyle s_{\hat{L}_1}= \sqrt{MSE\cdot \sum_{i=1}^t \dfrac{a_i^2}{n_i}}=\sqrt{0.0153\lrpar{\dfrac{1^2}{6} + 4\cdot \dfrac{(-1/4)^2}{6}}}\)
  • \(\displaystyle t = \dfrac{\hat{L}_1}{s_{\hat{L}_1}}\)
  • \(\displaystyle F = t^2 = (-3.7019)^2\)
  • \(\displaystyle SSC_L = \dfrac{\lrpar{\sum_{i=1}^t a_i\overline{Y}_{i\cdot}}^2}{\sum_{i=1}^t a_i^2/n_i} = (-0.2090)^2/\lrpar{\dfrac{1^2}{6} + 4\cdot \dfrac{(-1/4)^2}{6}}\)
With \(\alpha = 0.05\text{,}\) we reject \(H_0:L = \mu_1 - \dfrac{1}{4}(\mu_2 + \mu_3 + \mu_4 + \mu_5) = 0\) since \(|t|\geq t_{0.025,25} = 2.0595\text{.}\) We conclude that there is a significant difference of effectiveness between the control and average of all agents under \(\alpha = 0.05\)
