The structure of covariance for the model must be chosen specifically. We have four options; let \(0\lt \rho \lt 1\text{:}\)
  1. Variance Components (VC):
    \begin{equation*} \text{Cov}\,(Y_{i1k},Y_{i2k}) = \text{Cov}\,(Y_{i1k},Y_{i3k}) = \text{Cov}\,(Y_{i2k},Y_{i3k}) = 0 \end{equation*}
    VC is also referred to as "independence"; specifically, this means that there is no linear relationship between different random variables
  2. Compound Symmetry (CS):
    \begin{equation*} \text{Cov}\,(Y_{i1k},Y_{i2k}) = \text{Cov}\,(Y_{i1k},Y_{i3k}) = \text{Cov}\,(Y_{i2k},Y_{i3k}) = \rho\sigma^2 \end{equation*}
    We assume that each covariance is the same between different random variables
  3. First-Order Auto-Regressive, AR(1):
    \begin{align*} \text{Cov}\,(Y_{i1k},Y_{i2k}) = \text{Cov}\,(Y_{i2k},Y_{i3k}) \amp = \rho \sigma^2 \\ \text{Cov}\,(Y_{i1k},Y_{i3k}) = \rho^2\sigma^2 \amp \end{align*}
    AR(1) assumes that as measurements become further apart, extra factors of \(\rho\) are needed (eg, Time 1 and Time 2 only need \(\rho\text{,}\) but Time 1 and Time 3 need \(\rho^2\))
  4. Unstructured (UN):
    \begin{align*} \text{Cov}\,(Y_{i1k},Y_{i2k}) \amp = \sigma_{12} \\ \text{Cov}\,(Y_{i1k},Y_{i3k}) \amp = \sigma_{13} \\ \text{Cov}\,(Y_{i2k},Y_{i3k}) \amp = \sigma_{23} \end{align*}
    Unstructured assumes separate covariacne for each pair of repeated measurements.
