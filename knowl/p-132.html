From the weed control example from above, we have the following contrasts:
  1. Is there a difference of effectiveness between control and average of all agents?
    \begin{equation*} L_1 = \mu_1 - \dfrac{1}{4}(\mu_2 + \mu_3 + \mu_4 + \mu_5) \end{equation*}
  2. Is there an average effectiveness difference between biological and chemical agents?
    \begin{equation*} L_2 = \dfrac{1}{2}(\mu_2 + \mu_3) - \dfrac{1}{2}(\mu_4 + \mu_5) \end{equation*}
  3. Compare biological agents 1-2:
    \begin{equation*} L_3 = \mu_2 - \mu_3 \end{equation*}
  4. Compare chemical agents 1-2:
    \begin{equation*} L_4 = \mu_4 - \mu_5 \end{equation*}
It can be shown that all pairs of contrasts are orthogonal.
