We have some terms and values:
  • \(\sigma^2_\tau\) and \(\sigma^2_\varepsilon\) are called variance components
  • \(MSE = \hat{\sigma^2_\varepsilon}\) is the estimate of \(\sigma^2_\varepsilon\)
  • MSTrt is a point estimate of \(\sigma^2_\varepsilon + n\sigma^2_\tau\text{,}\) which we term EMS
  • \(\displaystyle MSTrt = \hat{\sigma^2_\varepsilon} + n\hat{\sigma^2_\tau}\)
  • Solving for \(\hat{\sigma^2_\tau}\text{,}\) we get \(\hat{\sigma^2_\tau} = \dfrac{1}{n} (MSTrt - \hat{\sigma^2_\varepsilon})\text{.}\) This is called the Method of Moments Estimation (MME) of \(\hat{\sigma^2_\tau}\text{.}\) Note that MME can yield negative variance estimates.
We can use specific techniques to avoid the negative variance component estimates; in SAS, PROC MIXED or (PROC GLIMMIX) statement allows us to use the method called Restriced Maximum Likelihood (REML) for that.
