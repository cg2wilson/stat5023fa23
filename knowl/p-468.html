The repeated measures can be analyzed using the following SAS code:
DATA one;
INPUT trt time expunit response;
DATALINES;
...
PROC MIXED DATA = one PLOTS=RESIDUALPANEL;
CLASS trt time expunit;
MODEL response = trt time trt*time/ DDFM = SATTERTH
REPEATED / SUBJECT = expunit(trt) TYPE = < option >
The line SUBJECT = expunit(trt) shows that each experimental unit is nested within treatment. The TYPE = < option > specifies covariance structure.
