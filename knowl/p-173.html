Recall that when we declare a signficance level \(\alpha\text{,}\) we declare the probability of making a type I error (rejecting the null when we should not). At the signficance level \(\alpha\text{,}\) we have
\begin{align*} \mathbb{P}(\text{at least one false rejection}) \amp = 1 - \mathbb{P}(\text{no false single rejection}) \\ \amp = 1 - (1-\alpha)^m \end{align*}
when we independently test \(m\) hypotheses. Note that we call the term
\begin{equation*} \mathbb{P}(\text{at least one false rejection}) \end{equation*}
the family-wise or experiment-wise error rate. So for example, if \(\alpha = 0.05\) and \(m = 10\text{,}\) the family-wise error rate is given by
\begin{equation*} 1 - (1-\alpha)^m \approx 1 - 0.599 = 0.401 \gt \alpha = 0.05 \end{equation*}
