Example 7.1.2.

An experiment studies the effects of four different pesticides on the yield of fruit from three different varieties of a citrus tree.
Source df SS MS \(F\) value \(Pr\gt F\)
Pesticide 3 2226.290 742.10 17.51 0.0001
Variety 2 3996.08 1998.04 47.13 \(\lt \) 0.00001
Interaction 6 456.92 76.15 1.80 0.1818
Error 12 508.67 42.39
Total 23 7187.96
There is no significant interaction; we interpret the main effects from the ANOVA table. We also reject the null hypotheses
\begin{equation*} H_0: \overline{\mu_{1\cdot}} = \cdots = \overline{\mu_{a\cdot}} \end{equation*}
and
\begin{equation*} H_0: \overline{\mu_{\cdot 1}} = \cdots = \overline{\mu_{ \cdot b}} \end{equation*}
We want to know which pairwise mean difference or contrast is significantly different from 0. This gives three questions:
  1. Let \(\overline{Y_{1\cdot\cdot}} = 53\text{,}\) \(\overline{Y_{2\cdot\cdot}} = 67.83\text{,}\) \(\overline{Y_{3\cdot\cdot}} = 51.17\text{,}\) and \(\overline{Y_{4\cdot\cdot}} = 73.83\text{.}\) Using Tukey’s HSD with \(\alpha = 0.05\text{,}\) identify which pair(s) of pesticides have significantly different average effect on yield.
  2. Given that \(\overline{Y_{\cdot 1\cdot}} = 46.88\text{,}\) \(\overline{Y_{\cdot 2\cdot}} = 59.25\text{,}\) and \(\overline{Y_{\cdot 3\cdot}} = 78.25\text{,}\) identify which pair(s) of varieties have significantly different average effect on yield using Tukey’s HSD with \(\alpha = 0.05\)
  3. Under \(\alpha = 0.05\text{,}\) test the following two null hypotheses using Scheffe’s Method:
    \begin{equation*} H_{01}: L_1 = (\overline{\mu_{1\cdot}} + \overline{\mu_{2\cdot}}) - (\overline{\mu_{3\cdot}} + \overline{\mu_{4\cdot}}) = 0 \end{equation*}
    and
    \begin{equation*} H_{02}: L_2 = 2\overline{\mu_{\cdot 2}} - (\overline{\mu_{\cdot 1}} + \overline{\mu_{\cdot 3}}) = 0 \end{equation*}
