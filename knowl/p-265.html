Due to the presence of significant action, we need to investigate the main effects of age for each product separately.
  1. Fix the first product \(B = 1\text{.}\) Then, consider the following pairwise comparisons for the main effect of age:
    \begin{align*} H_0: \amp \mu_{11} - \mu{21} = 0 \\ H_0: \amp \mu_{11} - \mu{31} = 0 \\ H_0: \amp \mu_{21} - \mu{31} = 0 \end{align*}
  2. Compare Tukey’s HSD with each pairwise mean difference.
    \begin{align*} |\overline{Y_{11\cdot}} - \overline{Y_{21\cdot}}| \amp = 3.3 \\ |\overline{Y_{11\cdot}} - \overline{Y_{31\cdot}}| \amp = 1.0 \\ |\overline{Y_{21\cdot}} - \overline{Y_{31\cdot}}| \amp = 2.3 \end{align*}
    and
    \begin{equation*} HSD = q_{0.05,6,54}\sqrt{147.1\cdot \dfrac{1}{10}} = 16.03 \end{equation*}
  3. We conclude that there is no main effect of age for the first product (breakfast cereals).
Now repeat the process with the second product \(B = 2\text{.}\)
  1. Consider the pairwise comparisons for the main effect of age:
    \begin{align*} H_0: \amp \mu_{12} - \mu{22} = 0 \\ H_0: \amp \mu_{12} - \mu{32} = 0 \\ H_0: \amp \mu_{22} - \mu{32} = 0 \end{align*}
  2. Compare Tukey’s HSD with each pairwise mean difference.
    \begin{align*} |\overline{Y_{12\cdot}} - \overline{Y_{22\cdot}}| \amp = 7.4 \\ |\overline{Y_{12\cdot}} - \overline{Y_{32\cdot}}| \amp = 22.5 \\ |\overline{Y_{22\cdot}} - \overline{Y_{32\cdot}}| \amp = 15.1 \end{align*}
    and
    \begin{equation*} HSD = q_{0.05,6,54}\sqrt{147.1\cdot \dfrac{1}{10}} = 16.03 \end{equation*}
    as before
  3. We conclude that there is a significant difference between the first and third age groups for the second product (video games).
