Another test is Bartlett’s test. The test statistic is given by
\begin{equation*} \chi^2 = \dfrac{1}{C}\left[ (n_\cdot - t)\log (MSE) - \sum_{i=1}^t (n_i-1)s_i^2\right] \end{equation*}
where
\begin{equation*} C = 1 + \dfrac{1}{3(t-1)} \lrpar{\sum_{i=1}^t \dfrac{1}{n_i-1} - \dfrac{1}{n_\cdot - t}} \end{equation*}
For this test, we don’t require the balanced case. Additionally, the test statistic approximately follows \(\chi^2_{t-1}\) under the null hypothesis. We set
\begin{equation*} H_0: \sigma_1^2 = \sigma_2^2=\cdots = \sigma_t^2 \end{equation*}
and reject if \(\chi^2 \geq \chi^2_{\alpha,t-1}\text{.}\)
