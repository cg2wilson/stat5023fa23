For interaction effects, we reject \(H_0:\text{ no interaction}\) when
\begin{equation*} F = \dfrac{MSAB}{MSSPE} \geq F_{\alpha,(a-1)(b-1),a(n-1)(b-1)} \end{equation*}
If the interaction is insignificant, we proceed to the main effects as follows:
  1. Reject \(H_0: \alpha_1 = \cdots = \alpha_a = 0\) if
    \begin{equation*} F = \dfrac{MSA}{MSWPE}\geq F_{\alpha, a-1,a(n-1)} \end{equation*}
  2. Reject \(H_0: \beta_1 = \cdots = \beta_b = 0\) if
    \begin{equation*} F = \dfrac{MSB}{MSSPE} \geq F_{\alpha,b-1,a(n-1)(b-1)} \end{equation*}
