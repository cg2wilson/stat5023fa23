In the previous example, note that there would be multiple \(a_i\) for the same comparison. For example, we can also set \(a_1 = 4, a_2=a_3=a_4 = -1\) for \(L_1\text{.}\) In either case, the test statistic is the same for any constant multiple of the original set of coefficients. To see this, given \(a_i\)s, let \(b_i = c\cdot a_i\) for some \(c\text{.}\) Then, we still obey the condition that
\begin{equation*} \sum_{i=1}^t b_i = 0 \end{equation*}
so we can say that
\begin{equation*} L' = \sum_{i=1}^t b_i\mu_i \end{equation*}
is a linear contrast. This means that the t-statistic for the contrast \(L'\) is given by
\begin{equation*} t = \dfrac{\sum_{i=1}^t b_i\bar{Y}_{i\cdot}}{\sqrt{MSE \sum_{i=1}^t b_i^2/n_i}} = \dfrac{c\sum_{i=1}^t a_i\bar{Y}_{i\cdot}}{\sqrt{MSE\cdot c^2\sum_{i=1}^t a_i^2/n_i}} = \dfrac{\sum_{i=1}^t a_i\bar{Y}_{i\cdot}}{\sqrt{MSE\cdot \sum_{i=1}^t a_i^2/n_i}} \end{equation*}
which is the same as for the constrast \(L\)
